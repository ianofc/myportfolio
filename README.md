# Portfólio de Ian Santos

Este é o portfólio pessoal de Ian Santos, desenvolvido com Flask, HTML, CSS e JavaScript. O projeto apresenta informações sobre mim, minhas habilidades, projetos e formas de contato.

## Tecnologias Utilizadas

- **Flask**: Framework web para Python.
- **HTML/CSS**: Estrutura e estilo das páginas.
- **JavaScript**: Interatividade nas páginas.
- **Tailwind CSS**: Framework CSS para estilização responsiva.
- **Font Awesome**: Ícones para a interface.
- **GitHub API**: Para buscar e exibir projetos do GitHub.
